# ecommerce-checkout-flowchart
Flowchart and pseudocode for a simple e-commerce checkout process. Microsoft Coding front end course Project. Coursera

## Objective
Demonstrate the logical flow of an e-commerce checkout process using a flowchart and pseudocode.

## Problem
Create a checkout process where the user:
- Selects products
- Adds them to the cart
- Enters shipping details
- Chooses a payment method
- Confirms the order

![Flowchart](flowchart.png)

## Pseudocode

```text
Start the purchase process

Allow the user to select products

Add the selected products to the cart

Prompt the user to enter shipping details

Prompt the user to select a payment method

If the user selects a digital payment platform
    Proceed to the digital payment process
Else if the user selects a credit card
    Proceed to the credit card payment process

Confirm the order

End the purchase process
```
